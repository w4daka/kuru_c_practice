# 苦しんで覚える C言語メモ

[これ](https://9cguide.appspot.com/index.html)の個人的なメモ

## C言語の構造

C言語のプログラムは、関数が集まって作られている。それに順番はない。

関数の構造

```c
型名　関数名(引数)｛処理｝
```

関数の命名規則

1. 半角アルファベット、数字、半角アンダースコア(\_)を使用可能
2. 1文字目には、数字をつかえない
3. あらかじめ決められた予約語は使用できない。予約済み識別子も名前として使用できない。

`引数`とは、関数に渡す数値の種類のこと

関数は、渡された数値を元に計算を行って結果を返すことができる。

## main関数は特別

C言語のプログラムは、main関数から実行される。main関数は、プログラムの入り口であり、必ず1つだけ存在する。

## C言語翻訳ソフト

高級言語はコンピュ－タには理解できないので、翻訳が必要。翻訳方法には、2種類あり、同時翻訳と、コンパイルがある。

同時翻訳はは、プログラムを実行しながら翻訳する方法で、インタプリタとも呼ばれる。

コンパイラは、C言語で書かれた文字列ファイルを、マシン語に翻訳するソフト。

コンパイルの流れ

```c
Ｃ言語のプログラム（ソースコード）
Ｖ
プリプロセス（文字列の調整）
Ｖ
コンパイル（翻訳）
Ｖ
リンク（結合）
Ｖ
実行可能なソフトウェア（実行可能ファイル）
```

## 数値の表示

文字列の中に`%d`指定子を使うと、その部分が数値に置き換えられるのです。
